This Website.

Although I was familiar with html and css before the class, I learned a lot about Javascript and PHP to help me develop this website. I also incorporated the bootstrap framework to help assist me in creating the website. I didnt host it online until much later.

JKUAT DEBATE SOCIETY WEBSITE

Towards the end of my second semester I embarked on creating an online platform for the JKUat Debate members to easily keep abreast with the club activities and for creation and distribution of content on an advanced level. Another objective in creating the system was to allow for more convinient member registration through the Website. visit at JKUAT DEBATE SOCIETY WEBSITE


This project mainly involved HTML5, Bootstrap, PHP, JS, MYSQLI and a bit of ajax.

MMTC Web-based Human Resource Management System

This has been among my most recent works; working collaboratively in a team of two. I was chiefly back end developer, mostly dealing in secure database communication in PHP and mysqlli. I was also involved in enforcing securty via both client_side validation(using scripts) and server side validation. Securing this system also included guarding against sql injections, directory traversals, denial of service and Man in the middle attacks.
